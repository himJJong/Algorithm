3) Like
where절과 함께 특정 패턴을 검색할 때 사용 ( 와일드카드 % : 0개 이상의 문자를 대체, _ 1개 이상의 문자를 대체 )

WHERE NAME LIKE '장%' - 장으로 시작하는 단어 검색

ex) 
Like 'a%' // a로 시작되는 모든것
Like 'a_%_%' // a로 시작되고 최소 3이상의 길이를 가진 것
LIKE '_a%'   // 두번째 자리에 a가 들어가는 모든 것
